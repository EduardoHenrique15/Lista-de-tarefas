# 📝 Lista de Tarefas (To-Do List)

Uma aplicação simples, moderna e responsiva de Lista de Tarefas desenvolvida com HTML, CSS e JavaScript puro (Vanilla JS). O projeto permite gerenciar tarefas diárias diretamente no navegador, com uma interface limpa e intuitiva.

## 🚀 Funcionalidades

- **Adicionar Tarefas:** Crie novas tarefas digitando no campo de entrada e clicando em "Adicionar" ou pressionando a tecla `Enter`.
- **Editar em Linha (In-line):** Altere o texto de uma tarefa existente diretamente na lista, sem pop-ups nativos do navegador. Inclui botões de "Salvar" e "Cancelar".
- **Remover Tarefas:** Exclua tarefas da lista instantaneamente clicando no botão "Remover".
- **Validações:** Impede a adição ou salvamento de tarefas em branco.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica da página.
- **CSS3:** Estilização moderna, layout responsivo e transições suaves.
- **JavaScript (ES6+):** Manipulação dinâmica do DOM e lógica de eventos.
